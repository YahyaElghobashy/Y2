# T106: QuickActionCard Component

Build the quick action card for the home dashboard.

## Read First
- `CLAUDE.md`
- `docs/DESIGN_SYSTEM.md`

## What to Build

File: `src/components/home/QuickActionCard.tsx`
Test: `src/__tests__/components/home/QuickActionCard.test.tsx`

A tappable card that links to a module. Used on the home dashboard in a 2x2 grid.

## Component API

```tsx
interface QuickActionCardProps {
  icon: React.ReactNode
  label: string
  description: string        // one-line subtitle
  href: string
  accentColor?: string       // optional override for icon background tint
  className?: string
}

export function QuickActionCard({ icon, label, description, href, accentColor, className }: QuickActionCardProps) { ... }
```

## Design Specs

- Container: `bg-elevated` (#FFFFFF), `rounded-xl` (12px), `shadow-soft`, padding `p-5`
- Full width (parent grid controls the column sizing)
- Icon area: 40px circle, `bg-accent-soft` (#E8D5C0) background, icon is 20px `text-accent-primary`
- Label: DM Sans, 15px, `font-semibold`, `text-primary`, `mt-3`
- Description: DM Sans, 12px, `text-secondary`, `mt-0.5`, single line, truncate if needed
- Hover: `scale(1.02)`, shadow expands slightly, 150ms ease-out (use Framer Motion `whileHover`)
- Tap: `scale(0.98)`, 100ms (use `whileTap`)
- Entire card is a Link

## Implementation Notes

- `"use client"` (Framer Motion needed)
- Wrap the Link in a `motion.div` for hover/tap animations
- Use `next/link`
- The icon circle could use a soft background tint — default is `accent-soft`, but `accentColor` prop allows overriding for visual variety

## Test Requirements

- Renders icon, label, description
- Links to correct href
- Accepts className

## Do NOT

- Don't add a border — shadow only
- Don't make it tall — compact card, not a hero
- Don't add an arrow or chevron — the whole card is the link
