## SettingsRow Component

A tappable row for settings navigation.

```tsx
interface SettingsRowProps {
  icon: React.ReactNode
  label: string
  subtitle?: string
  href?: string
  onClick?: () => void
  rightElement?: React.ReactNode  // e.g. toggle, badge
  destructive?: boolean           // red text for logout etc.
}

export function SettingsRow({ icon, label, subtitle, href, onClick, rightElement, destructive }: SettingsRowProps) { ... }
```

Design:
- Height: 56px, `px-6`, flex row, items-center
- Icon: 20px, `text-secondary` (or `text-red-500` if destructive)
- Label: DM Sans, 15px, `text-primary` (or `text-red-500` if destructive)
- Subtitle: 12px, `text-secondary`, below label
- Right side: either `rightElement` or a `ChevronRight` icon (12px, `text-muted`)
- Separator: 1px `border-border-subtle` between rows (not on the last one)
- Tap: subtle background highlight on press

If `href` provided, wraps in Link. If `onClick`, wraps in button.
