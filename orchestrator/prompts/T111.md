## Why This Exists

Faith is a genuine anchor in Yahya's life, not performative. He maintains 5 daily prayers, 3 pages of Quran daily, and morning azkar. The spiritual module will eventually be a quiet, focused space for tracking these practices — not a gamified streak counter, but something that feels like a prayer rug: simple, dedicated, sacred. For now, the shell communicates this reverence. The empty state should feel like a quiet room, not a broken feature.

---

## User Story

As Yahya, when I tap "Spirit" in the bottom nav, I arrive at what will become my daily practice space. For now it's empty but the copy signals what's coming — prayer times, Quran tracking, azkar checklists. The page feels calm and intentional. As Yara, I see the same shell — the spiritual features will be available to both users.

The copy: "a quiet space for what matters" — not "coming soon." The subtitle names the three core features (prayer, Quran, azkar) to set expectations.
