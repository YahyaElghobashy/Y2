## RelationshipTabs Component

A horizontal tab bar with 4 tabs. Active tab has a copper underline that slides (like BottomNav indicator).

```tsx
export function RelationshipTabs() { ... }
```

Tabs:
| Tab | Label | Icon (Lucide) | Empty State |
|-----|-------|---------------|-------------|
| notes | "Notes" | `MessageCircle` | "No notes yet" / "Write your first love note" |
| coupons | "Coupons" | `Ticket` | "No coupons yet" / "Create one for your partner" |
| coyyns | "CoYYns" | `Coins` | "CoYYns wallet empty" / "Start earning together" |
| send | "Send" | `Bell` | "Send a notification" / "Surprise your partner" |

Design:
- Tab bar: full width, `border-b border-border-subtle`, `px-6`
- Each tab: DM Sans, 13px, `font-medium`, `py-3`, equal width (flex-1)
- Inactive: `text-secondary`
- Active: `text-accent-primary`
- Underline: 2px, `bg-accent-primary`, slides with Framer Motion `layoutId="us-tab-indicator"`
- Tab content area: `px-6 py-6`
- Content crossfade: wrap in `AnimatePresence` with `mode="wait"`, fade transition 200ms

State: use `useState` to track active tab. Default: "notes".

- Page renders with PageHeader showing "Us"

- Don't pre-build any tab content вҖ” just EmptyState placeholders
