## HomeGreeting Component

A time-aware greeting at the top of the dashboard.

```tsx
export function HomeGreeting() { ... }
```

Logic:
- Before 12pm: "Good morning"
- 12pm-5pm: "Good afternoon"  
- 5pm-9pm: "Good evening"
- After 9pm: "Good night"

For now, hardcode the name as "Yahya" (will be dynamic after auth is built).

Design:
- Greeting: Playfair Display, 28px, `text-primary`, `font-bold`
- Date below: DM Sans, 14px, `text-secondary`, formatted as "Monday, March 2"
- Padding: `px-6 pt-6 pb-4`

## Widget Slots

Below the grid, add 2 empty placeholder divs styled as:
- `bg-elevated`, `rounded-xl`, `shadow-soft`, `p-6`, full width, `mx-6`
- Height: 100px
- Content: centered `text-muted` text saying "Widget coming soon"
- Gap between slots: `gap-4`
- These will later hold the CoYYns wallet, cycle tracker, etc.

## Wrap in PageTransition

The entire page content should be wrapped in `<PageTransition>` from `@/components/animations`.

## Implementation Notes

- `"use client"` (needs `Date` for greeting, uses client components)
- Import icons from `lucide-react`
- The page uses the existing app layout (no need to add AppShell here вҖ” it's in layout.tsx)
- Use `date-fns` `format()` for the date display: `format(new Date(), "EEEE, MMMM d")`
